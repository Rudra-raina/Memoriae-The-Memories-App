# 🍳 Foodle

Your personal recipe book on Android — save, search, edit, and share recipes with dish photos and external recipe discovery.

---

## 📱 Screen Recording

Watch the full app demo: https://drive.google.com/file/d/1giCuTXd67lgMbRIxx8OiTa7IZIj8yWPb/view?usp=share_link

---

## ✨ Features

- Add & save recipes with name, ingredients, steps, and a dish photo
- Capture or pick a photo from gallery to attach to any recipe
- Search and filter recipes by name or ingredient
- Edit or delete any saved recipe at any time
- Share recipes with friends directly from the app
- Browse external recipes fetched from an API

---

## 🛠️ Tech Stack

- Language: Kotlin
- UI: XML Layouts, RecyclerView, Fragments
- Local Storage: SQLite via Room Library
- Networking: REST API for external recipe data
- Media: Camera Intent and Gallery picker
- IDE: Android Studio

---

## 🚀 Getting Started

1. Clone the repo: git clone https://github.com/Rudra-raina/Foodle.git
2. Open in Android Studio via File > New > Import from Version Control
3. Let Gradle sync complete
4. Run on emulator or device (API 21+)

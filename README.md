# 🗺️ Memoriae — The Memories App

Your personal travel journal on Android — log trips, pin locations on a map, and relive your memories.

---

## 📱 Screen Recording

Watch the full app demo: https://drive.google.com/file/d/1qY5vCR3roBIhgQL2w2wKijfO7gFacH6C/view?usp=drive_link

## ✨ Features

- Create and save travel memories with title, description, and photos
- Pin memories to exact locations using Google Maps
- Browse all saved memories on an interactive map
- View memories as a journal-style list
- Firebase backend for real-time data storage and sync
- Camera and gallery integration for attaching trip photos

---

## 🛠️ Tech Stack

- Language: Kotlin & Java
- UI: XML Layouts, Fragments, RecyclerView
- Backend: Firebase Realtime Database
- Maps: Google Maps API
- Authentication: Firebase Auth
- Media: Camera Intent and Gallery picker
- IDE: Android Studio

---

## 🏗️ Architecture & Key Concepts

- Firebase Realtime Database for storing and syncing memories across devices
- Google Maps API for location pinning and interactive map browsing
- Firebase Authentication for user login and session management
- RecyclerView for rendering the memories list
- Camera and Gallery Intent for attaching photos to each memory

---

## 🚀 Getting Started

1. Clone the repo: git clone https://github.com/Rudra-raina/Memoriae-The-Memories-App.git
2. Open in Android Studio via File > New > Import from Version Control
3. Add your google-services.json from Firebase Console to the /app directory
4. Add your Google Maps API key to AndroidManifest.xml
5. Let Gradle sync complete, then run on emulator or device (API 21+)
